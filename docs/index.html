MM-CAD: A Multi-Modal CAD Dataset and Benchmark for Cross-Modal Geometric Learning

Anush Bharathi, Ananthakrishnan A, Ramanathan Muthuganapathy

Indian Institute of Technology Madras

Code 🤗 Dataset 📄 Paper (coming soon)

Abstract

We introduce MM-CAD, a large-scale multi-modal CAD dataset designed to level up retrieval and retrieval-augmented generation models for engineering geometry. MM-CAD comprises two complementary parts. MM-CAD:A brings 33,816 unique CAD models from eleven widely used benchmark datasets under a common identifier scheme, with isometric renderings, point clouds, human-curated multi-level text captions, and 3,276 real hand-drawn user sketches. MM-CAD:B curates 192,626 models from the 1M-model ABC corpus through a seven-stage pipeline centered on Manifold-Aware Adaptive Sampling (MAAS), which organizes models into semantically coherent neighborhoods rather than merely removing duplicates. Every retained model is annotated through a metadata-grounded pipeline that conditions caption generation on parsed construction sequences, producing photorealistic in-context images, contour sketches, and a hierarchical application taxonomy. We further introduce a joint retrieval architecture aligning text, sketch, image, B-Rep, and point cloud encoders in a shared Matryoshka embedding space, establishing the first unified cross-modal retrieval benchmark for large-scale CAD.

MM-CAD:B Curation Pipeline

Starting from the full 1M-model ABC corpus, a 7-stage pipeline produces MM-CAD:B:

  1. ResNet18 feature extraction + TSP-inspired pre-sorting for visual coherence
  2. Windowed 256-bit perceptual hash deduplication (O(n), removes 28.5%)
  3. ReCon++ encoder fine-tuned on MM-CAD:A drives two-stage hierarchical clustering
  4. Manifold-Aware Adaptive Sampling (MAAS) — 8 adaptive modes based on local curvature and surface classification
  5. Gemini 3 Pro/Flash caption generation from parsed FeatureScript construction sequences
  6. FLUX.2-Klein photorealistic image synthesis conditioned on CAD renders (~131K models)
  7. Recursive DPGMM taxonomy over BGE + co-occurrence keyword embeddings (4,862 nodes)

Benchmark

A joint multi-modal retrieval architecture aligns five modalities — text, hand-drawn sketch, isometric render, B-Rep solid, and point cloud — in a shared 768-dim Matryoshka embedding space. Pretrained checkpoints (tri-modal and 5-modal) are provided as reference baselines.

Key result: simple mean fusion of text + sketch + photorealistic image queries consistently outperforms every single- and pairwise-fusion baseline, establishing trimodal complementarity as an empirical fact for engineering CAD retrieval. Matryoshka truncation to d=128 retains 97.4% of d=768 R@1 performance.
